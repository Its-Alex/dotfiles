# Neovim Shortcuts

## Files

| Shortcut | Action |
|----------|--------|
| `Space f f` | Find file in project |
| `Space f g` | Search text in project (live grep) |
| `Space f b` | List open buffers |
| `Space f h` | Search in help |

> In Telescope: `Tab` to select multiple files, `Enter` to open.

## Buffers

| Shortcut | Action |
|----------|--------|
| `Shift+h` or `[b` | Previous buffer |
| `Shift+l` or `]b` | Next buffer |
| `Space b d` | Close buffer |
| `Space b D` | Force close buffer |

## Windows (splits)

| Shortcut | Action |
|----------|--------|
| `Space -` or `Space w -` | Split horizontal |
| `Space \|` or `Space w \|` | Split vertical |
| `Space w w` | Go to other window |
| `Space w d` | Close window |
| `Ctrl+h` | Move left |
| `Ctrl+j` | Move down |
| `Ctrl+k` | Move up |
| `Ctrl+l` | Move right |

## Native Neovim

| Shortcut | Action |
|----------|--------|
| `Y` | Yank to end of line |
| `[Space` | Add empty line above |
| `]Space` | Add empty line below |
| `[t` / `]t` | Previous / next tab |
| `[b` / `]b` | Previous / next buffer |
